# Data & AI World Cup 2026 Portfolio

This repository documents my learning journey from DataCamp's Data & AI World Cup event series.

The goal of this project is to demonstrate practical skills in data cleaning, soccer analytics, AI-assisted analysis, and AI agent development.

## Events Covered

| Date | Event | Focus |
|---|---|---|
| June 9, 2026 | How to Deal with Messi Data | Messy data, cleaning, exploratory analysis |
| June 10, 2026 | How Data and AI Are Changing Soccer Analytics | Soccer analytics, AI use cases, decision-making |
| June 11, 2026 | Build a Soccer Agent | AI agents, match data querying, soccer insights |
